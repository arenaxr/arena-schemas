
Goto URL
========


Goto given URL; Requires click-listener
Goto URL Attributes
-------------------

|Attribute|Description|Type|Default|Required|
| :--- | :--- | :--- | :--- | :--- |
|dest|dest|string; One of: ```['popup', 'newtab', 'sametab']```||Yes|
|on|on|string; One of: ```['mousedown', 'mouseup']```||Yes|
|url|url|string||Yes|
